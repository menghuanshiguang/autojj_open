# 基金周期低谷监控

> 桌面悬浮窗，智能识别处于周期性底部的基金，辅助定投和抄底决策。

## 使用方法

1. 下载 `FundMonitor.zip`（或直接下载三个文件）
2. 解压到任意目录
3. 双击 `FundMonitor.exe`

首次运行会拉取数据（约 40 秒），之后启动秒开缓存，每 6 小时自动后台刷新。

## 界面

- **左侧**：自选基金面板，点击 ☆ 添加关注
- **右侧**：筛选结果列表，支持点击表头排序、滚轮浏览
- **底部**：刷新 / 开机自启开关 / 下次刷新倒计时
- **托盘**：关闭窗口即隐藏到通知区域，右键可彻底退出

## 数据说明

数据来源为东方财富（天天基金），与官方 APP 一致。净值 T+1 更新。

## 文件说明

| 文件 | 用途 |
|------|------|
| `FundMonitor.exe` | 主程序 |
| `fund_cache.json` | 数据缓存（自动生成，删除可强制刷新） |
| `watchlist.json` | 自选基金列表（自动生成） |

## 常见问题

**为什么杀毒软件报警？**  
使用 Nuitka 编译为原生机器码，无恶意行为。添加信任即可。

**如何彻底退出？**  
右键任务栏托盘图标 → 退出程序。

**窗口拖到屏幕外了？**  
删除同目录下的 `fund_cache.json` 重新打开。
